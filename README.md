Quicksort algorithm for JavaScript.
See [docs](https://aureooms.github.io/js-quicksort).
Parent is [@aureooms/js-sort](https://github.com/aureooms/js-sort).

```js
let sort = quicksort.singletco( partition.hoare ) ;
```
